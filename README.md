# LinkedIn Quote Poster

Automatically generates inspirational quote images using AI and posts them to LinkedIn daily. Quotes are written by Claude, background scenes are generated by DALL-E 3 HD, and text is overlaid with Pillow. A web UI lets you manage everything from a browser.

![Dashboard preview](static/favicon.jpg)

---

## Features

- **AI-generated quotes** — Claude picks famous, real quotes matched to a mood/scene, never repeating one
- **AI-generated images** — DALL-E 3 HD creates a background scene that fits the quote's emotion
- **Clean text overlay** — Pillow renders pixel-accurate quote text with shadow and gradient directly on the image
- **Metadata stripping** — EXIF data is removed before posting
- **LinkedIn auto-posting** — Uses the latest LinkedIn Posts API (version 202503)
- **Web dashboard** — Generate, preview, post, view history, manage settings, and read logs from a browser
- **Daily scheduler** — Set a post time and the app handles the rest automatically
- **No quote repeats** — Every used quote is saved to a local JSON file

## Stack

| Layer | Tool |
|---|---|
| Quote generation | Anthropic Claude (claude-sonnet-4-6) |
| Image generation | OpenAI DALL-E 3 HD |
| Text rendering | Pillow (PIL) |
| Web UI | Flask + Waitress |
| LinkedIn API | OAuth 2.0 / Posts API v202503 |

## Setup

### 1. Clone and install dependencies

```bash
git clone https://github.com/yourname/linkedin-quote-poster.git
cd linkedin-quote-poster
pip install -r requirements.txt
```

### 2. Create your `.env` file

```env
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-proj-...
LINKEDIN_ACCESS_TOKEN=
LINKEDIN_PERSON_URN=
LINKEDIN_CLIENT_ID=
LINKEDIN_CLIENT_SECRET=
POST_TIME=09:00
```

### 3. Get a LinkedIn access token

You need a LinkedIn app with the **Share on LinkedIn** product enabled and these OAuth scopes:
`openid profile email w_member_social`

Then run:

```bash
python get_linkedin_token.py
```

Follow the browser prompt. Your `.env` will be updated automatically with `LINKEDIN_ACCESS_TOKEN` and `LINKEDIN_PERSON_URN`.

### 4. Linux font setup (if running on Linux / LXC / Docker)

Install fonts so Pillow can render text:

```bash
apt install -y fonts-liberation fonts-dejavu-core
```

Then update the `_FONT_CANDIDATES` list in `image_generator.py` to include Linux paths:

```python
_FONT_CANDIDATES = [
    "/usr/share/fonts/truetype/liberation/LiberationSans-Bold.ttf",
    "/usr/share/fonts/truetype/dejavu/DejaVuSans-Bold.ttf",
    "C:/Windows/Fonts/arialbd.ttf",  # Windows fallback
]
```

## Running

### Web UI (recommended)

```bash
python app.py
```

Open `http://localhost:5000` in your browser. Other devices on your local network can use `http://<your-ip>:5000`.

The app runs in production mode using Waitress (not Flask dev server).

### CLI (one post, no UI)

```bash
python main.py
```

### Headless scheduler (no UI)

```bash
python scheduler.py
```

## Deploying on Proxmox LXC

1. Create a Debian/Ubuntu LXC container with a bridged network interface
2. Copy your project files and `.env` into the container
3. Install system deps and Python packages:
   ```bash
   apt update && apt install -y python3 python3-pip fonts-liberation fonts-dejavu-core
   pip install -r requirements.txt
   ```
4. Update font paths in `image_generator.py` (see Linux font setup above)
5. Run `python app.py` — accessible at `http://<container-ip>:5000` from your network

To run it persistently, create a systemd service:

```ini
# /etc/systemd/system/linkedin-poster.service
[Unit]
Description=LinkedIn Quote Poster
After=network.target

[Service]
WorkingDirectory=/path/to/linkedin-quote-poster
ExecStart=python3 app.py
Restart=always

[Install]
WantedBy=multi-user.target
```

```bash
systemctl enable --now linkedin-poster
```

## Environment Variables

| Variable | Description |
|---|---|
| `ANTHROPIC_API_KEY` | Claude API key |
| `OPENAI_API_KEY` | OpenAI API key (DALL-E 3) |
| `LINKEDIN_ACCESS_TOKEN` | LinkedIn OAuth 2.0 access token |
| `LINKEDIN_PERSON_URN` | Your LinkedIn person URN (`urn:li:person:XXXX`) |
| `LINKEDIN_CLIENT_ID` | LinkedIn app client ID |
| `LINKEDIN_CLIENT_SECRET` | LinkedIn app client secret |
| `POST_TIME` | Daily post time in `HH:MM` format (default: `09:00`) |

## Security

- Never commit your `.env` file — add it to `.gitignore`
- The web UI is intended for local network use only, not public internet exposure
- Image metadata is stripped before every post
